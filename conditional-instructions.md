# Conditional Instruction Modules:

These file references are CONDITIONAL instructions and should ONLY be read when relevant to the
task(s) you are performing.

You SHOULD INFER their purpose from the file name and, if given, descriptions that follow.

When multiple areas apply, load all relevant files.

- `agent-specs.md`: whenever a "spec" is referenced
- `python/cli.md`
- `mise.md`: if a project has a `mise.toml`
- `python/uv.md`: if a project has `uv.lock` or `UV_*` environment variables in `mise.toml`
